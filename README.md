# Rice Agents

A modular, provider-agnostic framework for building and orchestrating AI agents.

## Documentation

See the [Documentation](docs/rice_agents/index.md) for detailed guides and API references.

## Examples

Check out the `examples/` directory for 17 comprehensive examples ranging from basic tools to massive 100-agent swarms.

## Quick Start

```bash
pip install rice-agents
```
